# 职引小程序 3.0 P0 开发进展

日期：2026-07-06

## P0 目标

将小程序从“求职工具合集”升级为高频打开的 AI 求职助手、校招日历与投递管理工具，先完成每日工作台、JD 快速匹配、投递追踪、校招筛选和我的页求职资产总览的最小闭环。

## 已完成

1. 首页首屏改为今日求职工作台，文案收敛为“登录生成你的求职计划”。
2. 首页金刚区扩展为两排核心入口：JD 匹配、校招日历、投递追踪、简历评分、AI 面试、AI 助手、面经题库、薪酬查询、机构测评、求职规划。
3. 新增小程序版 JD 快速匹配页面，支持输入 JD、生成匹配分、命中关键词、缺失关键词、优化建议和面试准备。
4. JD 匹配报告已保存到后端，支持历史报告列表、报告详情读取，并与投递追踪关联。
5. 投递追踪支持结构化保存 JD 匹配分、命中关键词、缺失关键词、AI 建议、下一步事项、简历版本和面试时间。
6. 投递追踪页新增记录详情弹层，可查看状态流、JD 匹配报告、下一步事项，并快速更新状态。
7. 校招日历已补齐届别、类型、地区、行业、学历、留学生友好、Sponsor/Visa、截止窗口筛选。
8. 校招列表和详情页支持收藏与提醒降级：订阅模板可用时请求微信订阅，不可用时保存站内或本地关注记录。
9. 我的页新增求职资产总览，统一展示简历、JD 报告、投递追踪、申请材料，并给出下一步行动入口。
