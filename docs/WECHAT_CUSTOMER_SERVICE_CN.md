# 微信小程序客服消息配置

更新时间：2026-06-25

## 微信公众平台后台

路径：

```text
微信公众平台 → 小程序后台 → 功能 → 客服
```

建议配置：

```text
1. 添加客服人员，使用客服微信扫码绑定。
2. 确认客服人员可以在微信客服/小程序客服工具中接收用户消息。
3. 如有自动回复能力，配置欢迎语：您好，这里是职引客服，请描述您遇到的问题，我们会尽快处理。
```

## 小程序内入口

已添加原生客服入口：

```text
package-user/pages/feedback/feedback
package-user/pages/about/about
```

用户点击「联系客服」或「在线客服」会通过 `open-type="contact"` 打开微信客服会话。

## 验证方式

1. 在微信开发者工具重新编译。
2. 进入「我的 → 意见反馈」，点击「联系客服」。
3. 进入「我的 → 关于我们」，点击「在线客服」。
4. 使用普通微信号在体验版中发送一条消息。
5. 确认后台绑定的客服人员能收到消息并回复。

## 注意

- 客服人员绑定必须在微信公众平台完成，代码无法代替扫码绑定。
- 没有绑定客服人员时，按钮仍会打开会话，但消息无法被人工及时处理。
- 客服消息适合处理账号、会员、投递、反馈和合作沟通；正式工单仍建议保留意见反馈表。
